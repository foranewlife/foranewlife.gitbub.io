Hexo NexT Three


JavaScript 3D library for NexT.

Preview

Install

npm install next-theme/hexo-next-three

Configure

# JavaScript 3D library.
# Dependencies: https://github.com/next-theme/hexo-next-three
three:
  enable: true
  defer: true
  cdn:
  waves:
    enable: false
    cdn:
  lines:
    enable: false
    cdn:
  sphere:
    enable: false
    cdn:
